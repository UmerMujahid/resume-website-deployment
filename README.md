## Project Overview
This repository contains a clean, responsive CV website that showcases my contact information, education, skills, projects/experience, and certificates. The site uses a simple, professional design.

## Features
- Header with name, title and portrait
- About section
- Education (chronological)
- Skills, Certificates, and Achievements
- Experience / Projects with technologies used
- Contact information and GitHub link
- Clean layout using Flexbox
- Fully responsive for desktop, tablet, and mobile via CSS media queries
